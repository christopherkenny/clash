# Get Gold Pass Season Information

Get Gold Pass Season Information

## Usage

``` r
coc_get_goldpass(key = get_clash_key())
```

## Arguments

- key:

  Required. API key. See
  <https://developer.clashroyale.com/#/documentation> Default:
  `cr_get_key()`

## Value

a `tibble` of gold pass info

## Examples

``` r
if (FALSE) { # clash::has_clash_key()
coc_get_goldpass()
}
```
